## 🎯 ¿Qué es Método Base?

Método Base es un software profesional diseñado específicamente para gimnasios que desean ofrecer planes nutricionales personalizados a sus clientes de manera rápida, científica y profesional.

### ✨ Características Principales

- 🔬 **Científicamente Validado**: Basado en la fórmula Katch-McArdle
- 🔄 **Rotación Inteligente**: Evita monotonía en los planes
- ⚡ **Rápido**: Genera planes en 3-5 minutos
- 📊 **Reportes Completos**: Estadísticas de tu gimnasio
- 📱 **Integración WhatsApp**: Envío directo de planes
- 🎨 **Personalizable**: Branding de tu gym en todos los PDFs

## 📖 Guía Rápida de Uso

### 1. Ingresar Datos del Cliente

Completa el formulario con:
- Nombre completo
- Teléfono (opcional pero recomendado para WhatsApp)
- Edad
- Peso en kg
- Estatura en cm
- Porcentaje de grasa corporal
- Nivel de actividad física
- Objetivo (déficit/superávit/mantenimiento)

**Validación en Tiempo Real:** Los campos muestran ✓ cuando son válidos.

### 2. Generar Plan

Click en "GENERAR PLAN Y PDF". El sistema:
1. Calcula metabolismo basal (TMB)
2. Calcula gasto energético total (GET)
3. Determina calorías objetivo
4. Distribuye macros
5. Selecciona alimentos con rotación
6. Muestra preview del plan

### 3. Confirmar o Modificar

En el preview puedes:
- Ver todas las comidas del día
- Verificar cantidades en gramos y equivalencias
- Confirmar para generar PDF
- Modificar datos si es necesario

### 4. Enviar al Cliente

El PDF se abre automáticamente. Puedes:
- Imprimirlo
- Enviarlo por WhatsApp (botón integrado)
- Guardarlo para historial

## 🆘 Solución de Problemas

### La aplicación no inicia

**Posibles causas:**
1. Licencia expirada o inválida
2. Archivo de licencia faltante
3. Windows Defender bloqueando

**Soluciones:**
1. Verificar que `licencia.lic` esté en el directorio
2. Contactar soporte para renovar licencia
3. Agregar excepción en Windows Defender

### Error al generar PDF

**Posibles causas:**
1. Sin permisos de escritura en carpeta
2. Espacio en disco insuficiente

**Soluciones:**
1. Ejecutar como administrador
2. Liberar espacio en disco
3. Verificar que carpeta "planes/" exista

### Base de datos corrupta

**Solución:**
1. Ir a Panel Admin > Base de Datos
2. Restaurar desde backup más reciente
3. Si no hay backups, contactar soporte
